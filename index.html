HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements.
  • If there are three elements nested inside of a "body" element, those three nested elements are knows as child elements.
  • The element that holds tall the children - in this case the "body" element - is knowns as the parent element.
  • When working with visible HTML elements, the content is contained between an opening tab and a closing tab.
  • The words "element" and "tag" are commonly used interchangeably in the web development world, but it is important to know the difference.
  • An element refers to the entire element from opening tag to closing tag.
  • Whereas tag refers only to what appears inside the angle body brackets.
  • The alt text is important for screen readers and other assistive technologies.
  • Alt text can also improve search engine rankings for your website.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes allow us to share a CSS rule to any element in the HTML, enabling us to freely choose which elements will be assigned the CSS rule.
  • The "rel" attribute specifies the relationship between the current document and the linked document resource.
  • The "href" attribute specifies the location (URL) of the external resource. The "href" attribute is using a relative path. This means the location is derived from the current location.
  • CSS Box Model: every HTML element is essentially a box. Each box consists of margins outside the element, and the element's borders, padding, and content.
  • A selector defines the element or attribute to which the rules, or declaration, will apply.
  • Declaration contains two important components - the CSS property we want to apply and the value of the property. Ex: the declaration color:blue; indicates the CSS property color and the value for that property, blue.
  • External CSS Style Sheet: the most common way to style a webpage. This is the preferred method because it allows developers to keep all their CSS rules in a spearate file, which makes design changes easier.

Git

  • Create Branch: git checkout -b BRANCH-NAME
  • Add Changes: git add -A
  • Check Status of location or file changes: git status
  • Configure User Name: git config --global user.name"[firstname lastname]"
  • Configure User Email: git config --global user.email "[valid-email]"
  • Save Modified & Staged Changes: git stash
  • Show All Commits in the Current Branch's History: git log

JavaScript

  • A variable is a name container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Array: a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • The location of data in an array is identified with a unique number called an index.
  • Use ".length" to get the length or to access the length of an array.
  • Events: are code structures that listen for activity in the browser, and run code in response.
  • The "alert()" function makes a pop-up box appear inside the browser window, but we need to give it a string as an argument to tell the function what mesage to display.
  • Arguments: a value (primitive or object) passed as input to a function.
  • Functions are a way of packaging functionality that you wish to reuse. It's possible to define a body of code as a function that executes when you call the function name in your code.
  • Conditionals: are code structures used to test if an expression returns true or not. A common conditional is the if...else statement
  • Data Types: String, number, boolean, array and object.
  • String: is a sequence of text known as a string. To signify that the value is a string, enclose it in single or double quote marks.
  • Number: is a number. Numbers don't have quotes around them.
  • Boolean: is a true or false value. The words true and false are special keywords that don't need quote marks.
  • Array: is a structure that allows you to store multiple values in a single reference.
  • Object: can be anything. Everything in JavaScript is an object and can be stored in a variable.
